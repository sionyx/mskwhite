Представь что ты руководитель разработки и перед тобой стоит задача по созданию бота для телеграм, который будет обрабатывать покупки с помощью Telegram Stars и выписывать ключи для vpn сервиса outline.

Бот должен быть написан на python.
Для работы с telegram и outline нужно использовать соответствующие библиотеки. 

Сценарий использования бота следующий:
- пользователь открывает бот и нажимает кнопку /start
- бот выдает приветственное сообщение и предлагает совершить покупку за 10 stars 
- пользователь оплачивает покупку
- бот подтверждает покупку сообщением
- бот обращается к сервису outline по api, заводит новую учетную запись с никнеймом пользователя из телеграм
- бот получает от сервиса outline ключ для доступа
- бот отправляет сообщение пользователю с ключом и ссылкой на скачивание клиента outline.

План по созданию бота должен предполагать ряд этапов, в каждом из которых бот будет получать новый функционал соответствующий пункту в сценарии использования. Нужно учесть, что каждый этап будет запрограммирован ии-агентов, поэтому в плане должны быть сформулированы соответствующие запросы к ии-агенту.

План нужно сохарнить в файл plan.md
